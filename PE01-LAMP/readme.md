# Servidor LAMP con Vagrant (PE01)

Este proyecto automatiza el despliegue de un entorno LAMP (Linux, Apache, MySQL, PHP) utilizando Vagrant y VirtualBox.

## Requisitos
- VirtualBox 7.0+
- Vagrant 2.4+
- 2 GB RAM disponibles

## Instalaci√≥n y Uso
1. Clonar el repositorio.
2. Abrir terminal en la carpeta `PE01-LAMP`.
3. Ejecutar `vagrant up`.
